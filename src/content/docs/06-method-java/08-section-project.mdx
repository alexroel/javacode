## Código de solución

```java

public class Aritmetica {

    static int sumar(int... numeros) {
        int suma = 0;
        for (int numero : numeros) {
            suma += numero;
        }
        return suma;
    }

    static int restar(int... numeros) {
        int resta = numeros[0];
        for (int i = 1; i < numeros.length; i++) {
            resta -= numeros[i];
        }
        return resta;
    }

    static double restar(double... numeros) {
        double resta = numeros[0];
        for (int i = 1; i < numeros.length; i++) {
            resta -= numeros[i];
        }
        return resta;
    }
}

```

public class PruebaAritmetica {

        double resultado2 = aritmetica.restar(10.5, 5.2, 2.1);

}
