En esta sección hemos aprendido sobre los bucles en Java y cómo se pueden utilizar para repetir una tarea varias veces. Los bucles son una estructura de control que nos permite ejecutar un bloque de código múltiples veces, hasta que se cumpla una condición.

## Resumen de los temas vistos

Hemos visto varios tipos de bucles en Java:

1. El bucle `for`: se utiliza cuando se conoce el número de iteraciones que se deben realizar.
2. El bucle `while`: se utiliza cuando no se conoce el número de iteraciones que se deben realizar, pero se conoce la condición de salida.
3. El bucle `do-while`: similar al bucle `while`, pero garantiza que el bloque de código se ejecute al menos una vez.
4. El bucle `for-each`: se utiliza para recorrer elementos de una colección o un arreglo.
5. El bucle `break` y `continue`: se utilizan para salir de un bucle o saltar a la siguiente iteración.
6. Operadores de incremento y decremento: `++` y `--`, que se utilizan para incrementar o decrementar una variable en una unidad.
