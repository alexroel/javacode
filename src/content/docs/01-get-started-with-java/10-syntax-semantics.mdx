Qué es la sintaxis y semántica de un lenguaje de programación? La **sintaxis** se refiere a las reglas que debes seguir al escribir código en un lenguaje de programación. Por ejemplo, en Java, debes terminar cada instrucción con un punto y coma (`;`). La **semántica**, por otro lado, se refiere al significado de las instrucciones que escribes. Por ejemplo, en Java, la instrucción `System.out.println("¡Hola, mundo!");` imprime el texto "¡Hola, mundo!" en la consola.

## Sintaxis de Java

La sintaxis de Java es similar a la de otros lenguajes de programación como C++ y C#. Aquí hay algunas reglas básicas que debes seguir al escribir código en Java:

- **Clases y métodos**: En Java, todo el código debe estar dentro de una clase. Cada programa en Java debe tener al menos una clase con un método `main` que es el punto de entrada del programa.
- **Llaves**: Las llaves `{}` se utilizan para delimitar bloques de código, como el cuerpo de una clase o un método.
- **Punto y coma**: Debes terminar cada instrucción con un punto y coma `;`.

Por ejemplo, aquí hay un programa simple en Java que imprime "¡Hola, mundo!" en la consola:

```java
public class HolaMundo {
    public static void main(String[] args) {
        System.out.println("¡Hola, mundo!");
    }
}
```

En este programa:

- `public class HolaMundo`: Define una clase llamada `HolaMundo`.
- `public static void main(String[] args)`: Define un método `main` que es el punto de entrada del programa.
- `System.out.println("¡Hola, mundo!");`: Imprime el texto "¡Hola, mundo!" en la consola.
- `System` es una clase que proporciona un objeto `out` que representa la consola.
- `println` es un método que imprime el texto en la consola.

## Semántica de Java

La semántica de Java se refiere al significado de las instrucciones que escribes. Por ejemplo, en Java, la instrucción `System.out.println("¡Hola, mundo!");` imprime el texto "¡Hola, mundo!" en la consola. Aquí hay algunos conceptos semánticos clave en Java:

- **Variables**: Las variables en Java tienen un tipo y un nombre. Debes declarar una variable antes de usarla.
- **Tipos de datos**: Java tiene tipos de datos primitivos como `int`, `double`, `boolean`, etc., y tipos de datos de referencia como `String`, `ArrayList`, etc.
- **Operadores**: Java tiene operadores aritméticos, de asignación, de comparación, etc., que se utilizan para realizar operaciones en las variables.
- **Estructuras de control**: Java tiene estructuras de control como `if`, `for`, `while`, etc., que se utilizan para controlar el flujo de un programa.

Por ejemplo, en el siguiente código Java, creamos una expresión matemática que se imprime en la consola:

```java

System.out.println("La suma de 2 + 3 es: " + (2 + 3));
```

En este código:

- `2 + 3` es una expresión aritmética que suma los números `2` y `3`.
- `"La suma de 2 + 3 es: " + (2 + 3)` es una expresión de cadena que concatena el texto "La suma de 2 + 3 es: " con el resultado de la suma.
