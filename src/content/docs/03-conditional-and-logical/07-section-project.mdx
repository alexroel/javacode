## Detalles del proyecto

1. Solicitar al usuario el monto total de consumo.
2. Aplicar descuentos segÃºn el monto de consumo:
    - Si el monto de consumo es mayor a S/.50 pero menor o igual a S/.100, aplicar un descuento del 10%.
    - Si el monto de consumo es mayor a S/.100 pero menor o igual a S/.200, aplicar un descuento del 20%.
    - Si el monto de consumo es mayor a S/.200, aplicar un descuento del 30%.
    - Si el monto de consumo es menor o igual a S/.50, no aplicar descuento.
3. Mostrar el monto total de consumo y el monto con descuento aplicado.

## Resultado esperado

El programa debe solicitar al usuario el monto total de consumo y mostrar el monto total de consumo y el monto con descuento aplicado.

```java

Monto de descuento aplicado: S/.30.00
Monto con descuento aplicado: S/.120.00
----------------------------------------
```

import java.util.Scanner;

public class SistemaDescuentos {
    public static void main(String[] args) {
        // Crear objeto Scanner
        Scanner leer = new Scanner(System.in);

        // Solicitar al usuario el monto total de consumo
        System.out.print("Ingrese el monto total de consumo: ");
        double monto = leer.nextDouble();

        // Calcular descuento
        double descuento = 0;
        if (monto > 50 && monto <= 100) {
            descuento = monto * 0.10;
        } else if (monto > 100 && monto <= 200) {
            descuento = monto * 0.20;
        } else if (monto > 200) {
            descuento = monto * 0.30;
        }

        // Calcular monto con descuento
        double montoConDescuento = monto - descuento;

        // Mostrar resultados
        System.out.println("----------------------------------------");
        System.out.printf("Monto total de consumo: S/. %.2f\n", monto);
        System.out.printf("Monto de descuento aplicado: S/. %.2f\n", descuento);
        System.out.printf("Monto con descuento aplicado: S/. %.2f\n", montoConDescuento);
        System.out.println("----------------------------------------");

        // Cerrar Scanner
        leer.close();
    }
}
