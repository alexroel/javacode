Los **operadores lógicos** en Java permiten combinar condiciones y expresiones booleanas para formar condiciones más complejas. Este artículo explora los operadores lógicos más comunes en Java, cómo funcionan y cómo podemos probarlos fácilmente utilizando **JShell**, la herramienta interactiva de Java.


## Operadores Lógicos  

Se utilizan para construir expresiones más complejas mediante combinaciones de condiciones.  

- **Operador**: `&&`
  - **Descripción**: AND lógico: true si ambas condiciones son true.
  - **Ejemplo**: `a > 0 && b > 0`

- **Operador**: `||`
  - **Descripción**: OR lógico: true si al menos una condición es true.
  - **Ejemplo**: `a > 0 || b < 0`

- **Operador**: `!`
  - **Descripción**: NOT lógico: invierte el valor booleano.
  - **Ejemplo**: `!a > 0`


**Ejemplo en JShell:**  

```java
int edad = 25;
boolean esEstudiante = true;

// AND lógico
edad > 18 && esEstudiante // Devuelve true porque ambas condiciones son true

// OR lógico
edad < 18 || esEstudiante // Devuelve true porque al menos una condición es true

// NOT lógico
!(edad > 18) // Devuelve false porque invierte el resultado de true a false
```


## Expresiones Combinadas  

Podemos mezclar operadores aritméticos, de comparación, lógicos y condicionales para crear expresiones más complejas.  

**Ejemplo en JShell:**  

```java
int a = 5;
int b = 10;

// Expresión combinada
boolean resultado = (a + b > 15) && (15 % b == 5);
```

**Explicación del ejemplo:**  

1. `(a + b > 15)` evalúa a `false` porque `5 + 10` es `15`, que no es mayor que `15`.
2. `(15 % b == 5)` evalúa a `true` porque `15 % 10` es `5`.
3. `(false) && (true)` resulta en `false` porque el operador `&&` requiere que ambas condiciones sean `true`.
