En esta sección, exploramos y dominamos los conceptos clave que forman la base de la lógica en programación. Aprendiste a utilizar operadores y estructuras condicionales para tomar decisiones dentro de tu código de manera eficiente y estructurada.  

## Lo que logramos en esta sección:  

- **Operadores de comparación**  
  Comprendiste cómo comparar valores y evaluar condiciones que producen resultados booleanos (`true` o `false`).  

- **Operadores lógicos**  
  Aprendiste a combinar múltiples condiciones utilizando `&&`, `||` y `!` para crear lógica más compleja.  

- **Operadores ternarios**  
  Simplificaste decisiones utilizando operadores ternarios, reduciendo el código repetitivo a expresiones compactas y elegantes.  

- **Estructuras de control condicionales**  
  Implementaste lógica condicional utilizando bloques `if`, `else if` y `else` para que tus programas reaccionaran de manera dinámica a distintos escenarios.  

- **Estructuras de control `switch`**  
  Manejar múltiples casos de forma ordenada fue más sencillo gracias al uso de la estructura `switch`.  

- **Proyecto práctico**  
  Finalizaste la sección desarrollando un sistema de descuentos para un restaurante. Este proyecto te permitió aplicar los conceptos aprendidos de manera práctica, resolviendo problemas reales con lógica condicional.  

Gracias a estos aprendizajes, ahora puedes diseñar programas que reaccionen a diferentes condiciones, optimizar tus decisiones en el código y construir soluciones dinámicas para problemas del mundo real. ¡Gran trabajo al completar esta sección!
