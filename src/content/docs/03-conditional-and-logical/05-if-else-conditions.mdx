Los condicionales son una estructura clave en cualquier lenguaje de programación, ya que permiten ejecutar bloques de código basados en condiciones específicas. En este artículo, exploraremos el uso de condicionales `if-else` en Java con el ejemplo proporcionado, abordando desde una condición simple hasta múltiples condiciones y anidación.


## Condiciones if-else
Las condiciones `if-else` son una estructura fundamental en programación que permite tomar decisiones en función de ciertas condiciones. La sintaxis básica de una declaración `if-else` en Java es la siguiente:

```java
if (condicion) {
    // Bloque de código si la condición es verdadera
} else {
    // Bloque de código si la condición es falsa
}
```


## Condición simple
La estructura básica de un condicional `if` evalúa una única condición. Si esta es verdadera, ejecuta un bloque de código.

**Ejemplo:**
```java
int numero = 0;

if (numero == 0) {
    System.out.println("El número es Cero");
}
```
- **Condición**: `numero == 0`
- Si se cumple, imprime: `"El número es Cero"`.


## Condición doble (`if-else`)
Cuando queremos realizar una acción diferente si la condición no se cumple, usamos `if-else`.

**Ejemplo:**
```java
int numero = 4;

if (numero > 0) {
    System.out.println("El número es positivo");
} else {
    System.out.println("El número no es positivo");
}
```
- Si `numero > 0`, imprime: `"El número es positivo"`.
- De lo contrario, imprime: `"El número no es positivo"`.

---

## Múltiples condiciones (`if-else if`)
Para manejar más de dos posibilidades, utilizamos `if-else if`.

**Ejemplo:**
```java
int numero = 0;

if (numero > 0) {
    System.out.println("El número es positivo");
} else if (numero < 0) {
    System.out.println("El número es negativo");
} else {
    System.out.println("El número es Cero");
}
```
- Evalúa:
  1. ¿Es positivo?
  2. ¿Es negativo?
  3. Si no es ninguna de las anteriores, entonces es cero.


## Anidación de condiciones
En algunos casos, se requiere evaluar una condición dentro de otra. Esto se conoce como **condicionales anidados**.

**Ejemplo:**
```java
int numero = 5;

if (numero > 0) {
    if (numero % 2 == 0) {
        System.out.println("El número es Par positivo");
    } else {
        System.out.println("El número es Impar positivo");
    }
} else if (numero < 0) {
    if (numero % 2 == 0) {
        System.out.println("El número es Par negativo");
    } else {
        System.out.println("El número es Impar negativo");
    }
}
```
- Primero, evalúa si el número es positivo o negativo.
- Luego, dentro de cada caso, determina si es par o impar.

## Buenas prácticas
1. **Claridad del código**: Usa comentarios y estructuras claras para evitar confusiones.
2. **Evitar anidaciones profundas**: Demasiados niveles pueden hacer el código difícil de leer. Se pueden simplificar usando operadores lógicos (`&&`, `||`) o funciones auxiliares.
3. **Pruebas exhaustivas**: Asegúrate de probar tu programa con casos límite (0, números negativos, números positivos).
