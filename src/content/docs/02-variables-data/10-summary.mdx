# Resumen

En esta sección, profundizaste en el uso de variables y tipos de datos en Java, estableciendo una base sólida para manipular y almacenar información en tus programas. Aprendiste cómo declarar y trabajar con variables, conocer los diferentes tipos de datos que Java ofrece, y cómo interactuar con el usuario para obtener y mostrar información. También aplicaste lo aprendido en un proyecto práctico para calcular el precio de venta de un producto.

## Lo aprendido

A lo largo de esta sección, descubriste:

- **¿Qué es una variable?**: Comprendiste que una variable es un contenedor que almacena datos en memoria y cómo utilizarla para manipular información en un programa.  
- **Variables en Java**: Exploraste cómo declarar, inicializar y utilizar variables en Java, entendiendo las reglas de nomenclatura y las mejores prácticas para escribir código claro.  
- **Tipos de datos primitivos**: Aprendiste sobre los datos básicos como `int`, `double`, `char` y `boolean`, y cómo usarlos para representar información elemental.  
- **Tipos de datos estructurados**: Descubriste datos más complejos como arreglos y objetos, que permiten manejar múltiples valores o estructuras más avanzadas.  
- **Conversión de tipos**: Dominas cómo convertir datos entre tipos compatibles (implícita y explícitamente), garantizando que tus programas sean precisos y funcionales.  
- **Entrada de datos**: Utilizaste la clase `Scanner` para capturar información ingresada por el usuario desde el teclado.  
- **Salida de datos**: Aprendiste a mostrar información en la consola utilizando métodos como `System.out.println()`, mejorando la comunicación con el usuario.  
- **Proyecto práctico - Precio de venta**: Aplicaste lo aprendido en un proyecto realista donde calculaste el precio final de un producto con base en el costo y un porcentaje de ganancia.

## ¿Qué puedes hacer ahora?

Gracias a lo aprendido, ahora eres capaz de:

- Declarar y usar variables de diferentes tipos, organizando tus programas de manera eficiente.  
- Trabajar con datos primitivos y estructurados, seleccionando el tipo adecuado para cada situación.  
- Realizar conversiones entre tipos de datos de forma segura, evitando errores comunes.  
- Capturar datos del usuario desde el teclado y mostrarlos de manera clara en la consola.  
- Resolver problemas prácticos, como el cálculo de precios, integrando lógica y manipulación de datos en tus programas.  

Esta base de conocimientos te permite avanzar con confianza hacia temas más avanzados como estructuras de control, métodos y programación orientada a objetos. ¡Felicidades por completar este paso importante en tu aprendizaje de Java!
