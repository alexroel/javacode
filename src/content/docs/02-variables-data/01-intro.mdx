¡Bienvenido a **Uso de Variables y Datos en Java**! En esta guía, aprenderás a trabajar con variables y diferentes tipos de datos en Java, entendiendo cómo manipular, almacenar y transformar información en tus programas. También explorarás la forma de interactuar con el usuario a través de entrada y salida de datos, y aplicarás lo aprendido en un proyecto práctico para calcular el precio de venta de un producto.  

## Objetivos de esta sección

El objetivo principal de esta sección es enseñarte a manejar datos de manera eficiente en Java, proporcionando las herramientas necesarias para trabajar con información de diferentes tipos. Al finalizar, podrás:  

- Declarar y utilizar variables en Java correctamente.  
- Trabajar con datos primitivos y estructurados según la necesidad.  
- Realizar conversiones entre diferentes tipos de datos.  
- Interactuar con el usuario mediante entrada y salida de datos.  
- Resolver problemas prácticos aplicando el uso de variables y lógica en Java.  

## Temas que exploraremos

En esta sección, aprenderás:  

### ¿Qué es una variable?
Descubre qué es una variable, cómo actúa como contenedor de datos y por qué es fundamental en la programación.  

### Variables en Java
Explora cómo declarar, inicializar y utilizar variables en Java siguiendo las reglas de sintaxis y mejores prácticas.  

### Tipos de datos primitivos
Familiarízate con los tipos de datos básicos de Java, como números enteros (`int`), decimales (`double`) y valores lógicos (`boolean`).  

### Tipos de datos estructurados
Aprende a trabajar con datos más complejos, como arreglos y objetos, que permiten almacenar y manejar múltiples valores.  

### Conversión de tipos
Domina las técnicas para convertir datos entre tipos compatibles, tanto implícita como explícitamente, evitando errores comunes.  

### Entrada de datos
Descubre cómo capturar datos ingresados por el usuario utilizando la clase `Scanner` de Java.  

### Salida de datos
Aprende a mostrar información al usuario de manera clara y organizada en la consola.  

### Proyecto práctico - Precio de venta
Aplica todo lo aprendido en un proyecto realista donde calcularás el precio final de un producto a partir de su costo y un porcentaje de ganancia.  

## Al finalizar esta sección

Al concluir esta sección, tendrás una comprensión sólida sobre cómo trabajar con variables y datos en Java. Serás capaz de:  

- Declarar y manejar variables de diferentes tipos en tus programas.  
- Elegir y usar el tipo de dato adecuado para cada situación.  
- Transformar datos entre distintos formatos de manera segura.  
- Crear programas interactivos que capturen y muestren información al usuario.  
- Resolver problemas prácticos utilizando variables y lógica en Java.  

¡Prepárate para dar el siguiente paso en tu camino como desarrollador y comienza a crear programas más robustos y funcionales! 🚀
