## Características de Go

<CardGrid stagger>
	<Card title="Robusto y Seguro" icon="seti:lock">
		Java tiene un sistema de gestión de excepciones y manejo de errores que ayuda a construir aplicaciones estables y seguras. Además, la gestión de memoria automatizada mediante recolección de basura (Garbage Collection) reduce el riesgo de errores de memoria.
	</Card>
	<Card title="Plataforma Independiente" icon="laptop">
		Gracias a la máquina virtual de Java (JVM), el código Java es independiente de la plataforma. Esto significa que un programa Java puede ejecutarse en cualquier sistema operativo que tenga instalada la JVM (principio de "write once, run anywhere" - "escríbelo una vez, ejecútalo en cualquier lugar").
	</Card>
	<Card title="Orientado a Objetos" icon="puzzle">
		Java sigue el paradigma de la programación orientada a objetos (POO), lo cual permite estructurar el código en objetos que representan tanto datos como el comportamiento, facilitando la reutilización y el mantenimiento.
	</Card>
	<Card title="Multihilo" icon="gitter">
		Java soporta el procesamiento multihilo, lo que permite ejecutar múltiples tareas de manera concurrente dentro de un mismo programa, mejorando así el rendimiento en aplicaciones que requieren multitarea.
	</Card>
</CardGrid>
